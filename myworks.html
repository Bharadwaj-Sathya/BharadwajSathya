Android Application for College Mangement System

The main aim of this project is to create an android app for managing different activities like updating Subjects, managing student & faculty department records in colleges.

The admin can access all the details of faculty, student. Summary details will be created and maintained for the students so that they can view their details like their subjects, marks details etc.

Managing all department, faculty, student’s information via smart phones.

This system provides a more security by providing a two step authentications .

Every user have a unique-id to login the respective role like student, faculty , and admin. Non sbitian or any other users cannot login in to the app which is more secured.

The proposed system has a CGPA calculator which a student can check semester wise. User friendly environment and anyone can understand easily.

There are 4 different modules in this project they are :

  • Information Module
  • Admin Module
  • Faculty Module
  • Student Module

Student-teacher communication :

The ideal concept of frequent student teacher interactions finds its space in the articles and books regarding performance enhancement but not in reality. Owing to the tight schedule of both the parties, it is just not possible to maintain the rapport through physical interactions. But now, we have apps.
